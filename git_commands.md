### Cloning a Github
    git clone <git_path>
    
### Initiating git
    git init
    
### Syncing data from github
    git pull
    
### Syncing data to github
    git push
    
### Adding/Stage files
    git add <files>
    
### Commiting changes
    git commit -m "msg"
    
### Reverting any specific file to specific commit point
    git checkout <commit hash> <filename>
